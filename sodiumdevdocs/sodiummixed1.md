# 构建sodium的开发环境

 * [1.安装IDE](#1)
 * [2.获取sodium](#2)
 * [3.正确安装合适的jdk-version](#3)
   
<h3 id="1">1.安装IDE</h3>
通常情况下，写mod所用的IDE可以是~~jb~~的IDEA，可以是Eclipse，也可以是微软的vscode，~~包括但不限于记事本~~。

更多详情请见fabric开发文档：

[fabric wiki](https://fabricmc.net/wiki/zh_cn:start)

本章以IDEA为例
下载IDEA：

[IDEA](https://www.jetbrains.com/zh-cn/idea/)

<h3 id="2">2.获取sodium</h3>
下载sodium源码，(本章以sodium1.16.5next为例)
sodium github仓库地址：

[sodium github](https://github.com/CaffeineMC/sodium-fabric)

~~git的话大佬自便~~

解压，备用。

<h3 id="3">3.正确安装合适的jdk-version</h3>

安装jdk:

[Oracle](https://www.oracle.com/java/technologies/downloads/)

Oracle的链接可能会很慢，建议用镜像站:

[华为jdk镜像站](https://repo.huaweicloud.com/java/jdk/)

输入<kbd>win</kbd>+R,打开cmd。
查看jdk版本
```
java -version
```
